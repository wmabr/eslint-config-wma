# WMA ESLint config

## Setup

### 1. Install dependencies:

```sh
npm i -D eslint @wmabr/eslint-config
```

### 2. Inside `eslint.config.js`:

```
import config from "@wmabr/eslint-config";

export default [
    ...config
];
```
